================================================================================
UNITY EDITOR SETUP STEPS
================================================================================

1. CREATE CANVAS
   - GameObject → UI → Canvas
   - Canvas Scaler: Scale With Screen Size
   - Reference Resolution: 1920x1080

2. CREATE MAIN CONTAINER
   - Right-click Canvas → Create Empty
   - Name: "MultiplayerMenu"
   - Add MultiplayerMenuUI.cs script
   - RectTransform: Stretch to fill

3. CREATE PANELS (One at a time)
   a) MainPanel
      - Right-click MultiplayerMenu → UI → Panel
      - Set anchor to stretch
      - Set active: TRUE

   b) CreateGamePanel
      - Same as above
      - Set active: FALSE

   c) BrowseGamesPanel
      - Same as above
      - Set active: FALSE

   d) StatusPanel
      - Same as above
      - Set active: FALSE
      - Make background semi-transparent

4. ADD BUTTONS
   - Right-click Panel → UI → Button - TextMeshPro
   - Set text, font size, colors
   - Add to Button's OnClick() in MultiplayerMenuUI script

5. ADD INPUT FIELDS
   - Right-click Panel → UI → InputField - TextMeshPro
   - Set placeholder text
   - Configure character limit

6. ADD TEXT ELEMENTS
   - Right-click Panel → UI → Text - TextMeshPro
   - Set font size, alignment, color

7. CREATE SCROLL VIEW
   - Right-click BrowseGamesPanel → UI → Scroll View
   - Delete Horizontal Scrollbar
   - Add Vertical Layout Group to Content
   - Child Force Expand: Width=ON, Height=OFF
   - Spacing: 10

8. CREATE LOBBY LIST ITEM PREFAB
   - Create from hierarchy template
   - Add LobbyListItem.cs script
   - Wire up all references
   - Drag to Prefabs folder
   - Delete from hierarchy

9. WIRE REFERENCES
   - Select MultiplayerMenu GameObject
   - Drag all panels to their fields
   - Drag all buttons to their fields
   - Drag input fields
   - Drag LobbyListItem prefab
   - Drag Content transform from ScrollView

10. TEST
    - Press Play
    - Click through each panel
    - Check Console for errors
